1. Адаптер.
patterns.lab2.shapes.Triangle - класс с треугольником, НЕ наследник GraphObject
patterns.lab2.shapes.TriangleAdapter - адаптер для использование этого класса

2. Компоновщик
patterns.lab2.shapes.ShapeGroups - класс для объединения фигур в группу, для возможности построить дерево элементов на сцене

3. Декоратор
patterns.lab2.shapes.DecoratorBase 	 - базовый класс декоратора
patterns.lab2.shapes.DecoratorNotes  - декоратор, дописывающий примечание для графического объекта на сцене
